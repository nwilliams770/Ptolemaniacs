Ptolemaniacs is an interactive visualization of the Ptolemaic Dynasty family tree. It is a force-directed graph that uses JavaScript to render the parental, spousal, and succession relationships based on various, user-controlled filters.

Created exclusively in JavaScript, utilizing the d3 library created by Mike Bostock.
